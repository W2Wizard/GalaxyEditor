# About
An opensource tool that lets you modify the galaxy graph or 'Holy'graph of Intra (V2).

## Setup

As of now building is a bit rough ...

```bash
npm install --prefix ./backend
npm install
```

Start the backend:
```bash
npm run start --prefix ./backend
```

Start the frontend:
```bash
npm run start
```
